# FDM Monster CLI

This tool @fdm-monster/cli will help to install and update the FDM Monster CLI on any system equipped with Node 16.

## Installing the CLI

> npm i -g @fdm-monster/cli

Or the full form of this alias:

> npm install --global @fdm-monster/cli

## Checking it works

> fdm -v

Running this command should give you this output: 
>    FDM Monster CLI version vx.y.z
